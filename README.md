# git-tidy

## Compatibility

`git-tidy` is compatible with Python 3.9 - 3.13.

## Documentation

[View the git-tidy docs here](https://git-tidy.readthedocs.io/)

## Installation

Install `git-tidy` with:

    pip3 install git-tidy
